Models: inspecting your results

This section describes the elements in the results section to allow you to modify, and inspect the results.

Scene selector

Use the scene selector drop down menu to switch between the two view styles:

Spatial, angle, and wavelet

When a synthetic model is generated a figure will appear in the results area showing 3 variable intensity images through the model’s domains: a spatial cross-section (t-x domain); an angle gather (t-theta domain); and a wavelet gather (t-F domain).

Spatial only

Single panel showing a cross-section view of the synthetic data, with an isolated wiggle plot track of the selected trace.

Parameter control panel

Scene selector – Choose between a cross-section view, or view of the cross-section, angle gather and frequency gather with instantaneous amplitude plots.

Trace –Move the position of the wiggle trace along the spatial cross-seciton.

TWT –Move the vertical position of the horizontal green location of corresponding to the instantaneous amplitude plots.

Gain slider –Since Modeler, uses normalized wavelets, the synthetic data is a direct band-limited representation of the reflectivity of the earth model. The gain of the seismic display may need to be increased to make small reflectivity contrasts visible. Values range between 0 and 10. The default is 1.

Offset angle –Change the offset angle displayed on the cross-section and frequency gather panels. The default is zero; normal incidence. Change this slider to simulate the appearance of partial offset stacks.

Frequency – Modify the frequency content of your wavelet. For Ricker wavelets, the range of center frequencies is 8-85 Hz. Please note that center frequency is not the same as bandwidth. See Agile blog post: F-is-for-Frequency.

Phase – Rotate the phase of the model wavelet. The default, which is set to zero, represents a maximum positive value for downward increase in impedance across an interface.

Signal:noise (dB) – Add noise to your model by modifying the signal to noise ratio slider. Noise is applied in the frequency domain in proportion to the bandwidth of the signal.

Wavelet selector – Select from either Ricker or Ormsby wavelets. The Ricker wavelet is defined with a single center frequency using the Frequency slider.

Using the instantaneous amplitude graphs

Each domain has a corresponding area below the for plotting instantaneous amplitude values indicated by the horizontal green line. This is akin to an amplitude extraction at a specific two-way time. The instantaneous amplitude is plotted as a function of position, angle, and centre frequency for the spatial cross-section, angle gather, and wavelet gather respectively.

Use the TWT slider to change the position of the instantaneous amplitude plots to inspect:

Time slices

Functions of instantaneous amplitude versus position represents an amplitude extraction along a time-slice.

AVO response

Instantaneous amplitude curves in the angle domain represent AVO response.

Turning response

Instantaneous amplitude plots in the frequency panel represent turning curves

Rock property table

The elastic properties of each rock contained within the earth model is returned in a table with SI units. The computed rock properties include:

  • Acoustic impedance
  • Poisson’s ratio (PR)
  • Young’s modulus (E)
  • Lame’s first parameter (λ)
  • Shear modulus (μ)
  • Bulk modulus (K)
  • P-wave modulus (M)

You can export this table to a CSV (comma separated value) file for further analysis and using in other programs.
