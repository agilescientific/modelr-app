Models: inspecting your results

This section describes the elements in the results section to allow you to modify, and inspect the results.

Scene selector

When a synthetic model is generated a figure will appear in the results area showing 3 variable intensity images through the model’s domains: a spatial cross-section (t-x domain); an angle gather (t-theta domain); and a wavelet gather (t-F domain).

Spatial, angle, and wavelet

3 panels

Spatial only

Single panel showing a cross-section view of the synthetic data, with an isolated wiggle plot track of the selected trace.

Wiggle trace

The wiggle It is used to serve as a reference marker. Please note that it is the same trace for all panels. In some cases, the wiggle trace will appear wider or narrower within the wavelet gather. This happens because the wavelet gather increases in powers of 2 (not linearly), which causes some non-linear amplification when plotting the wiggle trace.

Offset angle slider

Use the Offset angle slider to move the position of the wiggle trace on the angle gather.

Frequency slider

Use the Frequency slider to change the center frequency of the wavelet shown in the spatial cross-section and angle gather.

Instantaneous amplitude graphs

Each domain has a corresponding area below the for plotting instantaneous amplitude values indicated by the horizontal green line. This is akin to an amplitude extraction at a specific two-way time. The instantaneous amplitude is plotted as a function of position, angle, and centre frequency for the spatial cross-section, angle gather, and wavelet gather respectively.

Use the TWT slider to change the position of the instantaneous amplitude plots to inspect:

Time slices

Functions of instantaneous amplitude versus position represents an amplitude extraction along a time-slice.

AVO response

Instantaneous amplitude curves in the angle domain represent AVO response.

Turning response

Instantaneous amplitude plots in the frequency panel represent turning curves

Using data sliders to modify model parameters

Gain slider

If you can see individual pixels, the image probably doesn't have enough resolution and your seismic experiment will suffer from sampling artifacts.

Frequency

By default, the number of traces in the synthetic will be the same as the width of the image in pixels. As a result, one way to add more traces to your synthetic model is to boost the resolution of your input image. Note that large images will be slower to process.

Phase

The vertical scale is set after upload, so the aspect ratio can be essentially anything that allows you to achieve the information density and geological complexity that you want to model.

Noise

Add noise to your model by modifying the signal to noise ratio slider. Noise is applied in the frequency domain in proportion to the bandwidth of the signal.

Wavelet

Select from either Ricker or Ormsby wavelets. The Ricker wavelet is defined with a single center frequency using the Frequency slider.

Rock property table

A table of elastic rock properties in SI units including:

  • Acoustic impedance
  • Poisson’s ratio (PR)
  • Young’s modulus (E)
  • Lame’s first parameter (λ)
  • Shear modulus (μ)
  • Bulk modulus (K)
  • P-wave modulus (M)

You can export this table to a CSV (comma separated value) file for further analysis and using in other programs.
