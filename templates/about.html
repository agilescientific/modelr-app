Modelr builds simple forward models of reflection seismic responses. In other words, you give it some geology, and it tries to predict what seismic reflection data would look like.

Why? We think forward models are the most important thing that nobody does, because the tools are esoteric and expensive. Well, they were.

The geeky details

This web app is built in Python on the open source goodness of NumPy, SciPy, and lots of other libraries, including our own agilegeo geophysical utility library.

The front end is built on Python, with a bit of JavaScript, using the amazing Twitter Bootstrap front-end framework. The app server is built in Python with the Jinja templating framework, and runs on Google App Engine.

The back end Modelr app, where all the science lives, is a custom-built WebApp2 server and runs on Amazon EC2. The back end is entirely open source and can be found on GitHub.

The first version of the app was built by Matt and Sean Ross-Ross in the spring of 2012.
