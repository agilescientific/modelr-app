What is modelr?

What? This web application builds simple forward models of reflection seismic responses.

How? There are two ways to use the app: go to the home page and use the form; or use the web API.

Why? We think forward models are the most important thing that nobody does, because the tools are esoteric and expensive.

Credits

Built with love by @kwinkunks (aka Matt Hall), @EvanBianco, and Ben Bougher in the autumn of 2013.

The first version of the app was built by Matt and Sean Ross-Ross in the spring of 2012.

This web app is built in Python on the open source goodness of NumPy, SciPy, and agilegeo, and lots of other libraries. It was developed in Enthought Canopy and depends on the Canopy CLI. It is running on the amazing Google App Engine and Amazon EC2.
